Overview

This guide will help you analyze your retirement planning batch results using Google Sheets. The CSV file from your batch calculation contains comprehensive year-by-year projections for multiple retirement scenarios.

Step 1: Import Your CSV Data

  1. Open Google Sheets: sheets.google.com
  2. Click File â†’ Import
  3. Click Upload tab
  4. Drag your CSV file or click Browse
  5. Select "Replace spreadsheet"
  6. Click Import data

Step 2: Understanding Your Data

Input Parameters (Columns A-N)

Column Description
scenario_id Unique identifier for each scenario
retirement_age Age when you retire
rrsp_balance RRSP/RRIF starting balance
tfsa_balance TFSA starting balance
num_pensions Number of pension income sources
num_properties Number of real estate holdings

Results (Columns O+)

  • year/age: Timeline of your retirement
  • total_balance: Combined value of all accounts
  • gross_income: Total annual income
  • taxes_estimated: Estimated taxes paid
  • success: TRUE if money lasts, FALSE if depleted
  • final_balance: Money remaining at life expectancy

Step 3: Find Your Best Scenarios

  • Buffer Amount: How much cushion do successful scenarios have?
